¿Cómo funciona?
1
El sistema corre solo — no tienes que hacer nada
Cada hora, una tarea automática ejecuta el runner, actualiza los datos y regenera los dashboards HTML. Solo abre el HTML que quieras ver y presiona F5 para actualizar.
Automático — cada hora
2
Para explorar ciencias nuevas — pídele a Claude
El mapa de cierres se actualiza cuando Claude hace búsquedas reales en bases de datos académicas. Di: "actualiza el mapa de cierres" o "explora [disciplina]" y lo hace al momento.
Manual — cuando quieras
3
Los tres archivos que importan
Solo necesitas abrir los 2 HTML y el informe Word. Los archivos .py y .json son el motor — se actualizan solos y no necesitas tocarlos.
Los .py y .json se actualizan automáticamente
